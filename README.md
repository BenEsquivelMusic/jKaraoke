# jKaraoke

Java FX Desktop application for playing karaoke media.

## Features
- Karaoke media player
- Singer Queue management
- Equalizer with presets
- Settings for customization

## Build Instructions

To build the application, ensure you have **Java 25** and **Maven** installed.

1. Open a terminal in the `application` folder.
2. Run the following Maven command:

```bash
mvn clean test javafx:jlink
```
