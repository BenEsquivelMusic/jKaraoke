Welcome to jKaraoke

jKaraoke is a robust JavaFX desktop application designed for managing and playing karaoke media. Whether you are hosting a party or managing a professional karaoke night, jKaraoke provides the tools you need to queue singers, manage songs, and control playback seamlessly.

Key Features

  • Media Support: Plays standard media files and supports CDG (CD+Graphics) for on-screen lyrics.
  • Queue Management: Easily add singers, re-queue them, or modify their song choices on the fly.
  • Event Management: Save and load your karaoke sessions ("Events") to resume later or keep records.
  • Audio Control: Built-in 10-band Equalizer with presets to fine-tune your sound.
  • Dual Screen Support: Dedicated Karaoke Media View window for displaying lyrics on a secondary monitor.
